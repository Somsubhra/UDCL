UDCL
====

Urban Dictionary Command Line Utility

### Quick install
Run the following on terminal
```
curl -s https://raw.githubusercontent.com/Somsubhra/UDCL/master/install.sh | sudo sh
```
### Usage
```
udcl <word>
```

### Un-install
Run the following on terminal
```
sudo rm -rf /usr/bin/udcl
```
